jetson nano刷机

Ⅰ. 需要准备的材料

  1. Jetson nano 开发套件，厂家标配；
  2. MicroSD 卡，至少 16GB 吧；
  3. 5V, 2A 电源。

Ⅱ. 下载jetson nano系统镜像

官网下载地址： Jetson Nano Developer Kit SD Card Image

Ⅲ. 格式化SD卡

利用 [SD Card Formatter](SD Card Formatter) 进行格式化即可，如下图所示：

这里的 Volume label （卷标）请留空！

点击 Format 即可快速格式化SD卡。

Ⅳ. 镜像文件写入SD卡

下载，安装和启动 Etcher

注：这里还有便携式版本，请不要下载，利用便携式版本写入镜像文件会报错。

①选择镜像文件，就是第二步下载好的。

②选择烧写的设备，就是电脑的SD卡。

③点击，等待即可烧写完毕。

注：烧写镜像的时候可能会出现格式化卷宗的提示，请不要格式化，直接点取消即可。这是因为烧录成功后系统可能会因为无法识别分区而提示格式化分区，linux下的磁盘分区win下是看不到的。

Ⅴ. 将SD卡插入jetson nano

这里介绍带显示器的安装方法，无头模式安装不推荐，后序介绍~

jetson nano 的底板资源请参考 JETSON NANO DEVELOPER KIT

图片来源：Setup and First Boot

注意：jetson nano 支持五种供电方式，如果采用 DC 直流电供电方式，需要短接J48跳线帽，如果使用 microUSb 则不需要。

（microUSb就是安卓数据线）

如何短接J48跳线帽，首先应该仔细阅读前面的底板资源文档，找到J48跳线帽在哪，会提供一个跳线帽，把这两个接口套上就好了。

通电，等待一小会显示器即可显示安装 Ubuntu 界面。

安装完 Ubuntu 看到如下界面，恭喜，刷机成功~

图片来源：Setup and First Boot

Ⅵ. 无头模式安装

找一个安卓数据线，一端连接电脑 usb 口，一端连接 jetson nano 的 microUSb 口。

等待会，此电脑会多出个盘，打开，里面有用各种模块连接的信息文档。

选择一个最简单的，利用 PuTTy 进行连接。

打开 README-usb-dev-mode.txt 文件，找到 jetson nano 的 ip

打开 PuTTy 输入 ip 地址即可进行安装系统引导，只不过这个安装引导是没有图形化界面的。

安装完系统后，也可以通过PuTTy 进行连接。

Ⅶ. 总结

初次刷机还是十分简单的，就是有时候看的英文文档有点头痛，还是要多记单词才行。

本文在无头模式连接上只介绍了通过 PuTTy 进行连接，实际上还支持 VNC 和 wifi 进行连接的，大致原理一样，阅读相应的文档即可解决。

Ⅶ. 更多资源

  1. Nvidia官网 所有关于 jetson nano 板子的介绍都有，不过基本上都是英文的。
  2. JETSON NANO DEVELOPER KIT 这个是前面说的 jetson nano 的底板资源介绍。
  3. yahboom 亚博智能官网，有各种各样的 Demo 和学习资源。
  4. Jetson Nano刷机教程 CSDN 上的刷机教程，除此之外，还多了个保护套件的说明。
